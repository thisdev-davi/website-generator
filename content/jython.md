# Jython

Jython is an implementation of the Python programming language designed to run on the **Java platform**.

## Key Features

*   **Java Integration**: Seamlessly import and use Java classes within Python scripts.
*   **Dynamic**: It offers the dynamic capabilities of Python while running on the robust JVM.
*   **Library Access**: Access to the vast ecosystem of Java libraries.

## Example Usage

Here is a simple example of how you might mix Java and Python:

```
from java.util import Date

d = Date()
print("Current date is:", d)
```

## Why use Jython?

1. You want to script an existing Java application.
2. You want to use Python syntax but need Java libraries.
3. You are testing Java code using Python test frameworks.
